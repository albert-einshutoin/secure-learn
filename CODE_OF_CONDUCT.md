# Code of Conduct

## Our Standards

Participants are expected to:

- Be respectful and constructive.
- Keep security research inside authorized environments.
- Avoid harassment, discrimination, and personal attacks.
- Share findings in a way that helps others learn safely.

## Unacceptable Behavior

Unacceptable behavior includes:

- Using this project to target systems without authorization.
- Publishing real secrets, private data, or exploit results from third-party systems.
- Harassment, abusive language, or sustained disruption.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict
participation when behavior conflicts with this code of conduct.
