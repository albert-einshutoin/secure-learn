WhitehatSREBackend

到達目標

  • 第三者環境を攻撃しない制約を説明できる
  • clone、branch、test、PR、CI確認までを一人で実行できる
  • レポートやログから秘密情報を除外できる

カバー領域

  • 許可範囲
  • GitHub Flow
  • Docker基礎
  • 証跡管理

Hands-on Flow

  • README、SECURITY、CONTRIBUTINGを読み、許可された閉域ラボだけを対象にする
  • feature branchを作り、ローカル品質ゲートの軽量チェックを実行する
  • incident reportとvulnerability remediation PRのテンプレートを埋める

Docker実行

scripts/learning_phase.sh config p0
scripts/learning_phase.sh start p0
scripts/learning_phase.sh status p0

実行コマンド

git checkout -b training/p0-ethics-setup
docker compose config -q
scripts/scenario_html_check.sh
scripts/learning_phase.sh guide p0

合格証跡

  • 対象、日時、実行コマンド、影響範囲を記録している
  • 秘密情報や外部ターゲットを含まない学習ログになっている
  • GitHub FlowでPR化できる
