総合評価

現在のSecure Learnは、ネットワーク偵察、認証攻撃、SQLi修正、OS監査、SLO/incident drill、Kubernetes基礎までを持つため、実務中級から上級入口の教材として成立しています。世界レベルを名乗るには、API business logic、cloud/IAM、observability、supply chain、release engineeringをさらに実測化する余地があります。

  • API Securityの上位領域では、IDOR/BOLA、SSRF、unsafe file upload、CSRF/CORS、business logic abuseを追加すると世界レベルに近づく。
  • SRE領域では、burn-rate alert、metrics/traces、canary、rollback drill、backup/restore、capacity planningを実測化する余地がある。
  • Backend領域では、schema validation、pagination、migration/rollback、transaction、concurrency、contract compatibilityをさらに深掘りできる。
  • Cloud領域では、IAM/RBAC、Ingress/TLS、image scan、SBOM、secret rotation、policy-as-codeをCIへ組み込む余地がある。

S1 ポートスキャン L3/L4 Whitehat, SRE 3/5 実務中級 nmapで露出ポートを列挙し、Suricata/Kibanaで偵察を検知する基礎シナリオ。

S3 SQLインジェクション L7 Whitehat, Backend 4/5 実務上級 sqlmap/curlでSQLiを試行し、アプリが400で拒否し、DB queryがparameterizedであることを証明する。
S4 DoS攻撃 L7/SRE SRE, Whitehat, Backend 3/5 実務中級 安全なリクエスト量でL7負荷をかけ、rate limit、latency、availability、BAN判断を確認する。
S5 重要ファイル改変 OS Whitehat, SRE 3/5 実務中級 Auditdで重要ファイル変更を記録し、変更者、時刻、対象、復旧判断を追う。
S6 権限昇格 OS Whitehat, SRE 3/5 実務中級 sudo/su/SUID探索をAuditdで記録し、最小権限と異常昇格の判断を学ぶ。
S7 横断インシデント 全体 Whitehat, SRE, Backend 4/5 実務上級 偵察、認証攻撃、SQLi試行、ファイル/権限イベントを一つのattack chainとして相関する。
S8 L2 ARP観測 L2 Whitehat, SRE 2/5 基礎良好 Docker bridge上のARP/neighbor cacheを観測し、L2の限界と証跡不足を説明する。
S9 L3 ICMP到達性・偵察 L3 Whitehat, SRE 3/5 実務中級 ping/tracerouteで到達性を確認し、ネットワーク障害とアプリ障害を切り分ける。
S10 L4 TCP状態・フラグ異常 L4 Whitehat, SRE 3/5 実務中級 SYN/connect/FIN/NULL/XMAS scanの違いを観測し、L4状態と検知ルールを理解する。
S11 L5セッション圧迫 L5 SRE, Backend 3/5 実務中級 不完全HTTP sessionを保持し、timeout、connection limit、SLO影響を観測する。
S12 L6 TLS可視性境界 L6 Whitehat, SRE, Backend 2/5 基礎良好 TLS ClientHelloをHTTP serviceへ送り、暗号化境界とIDS可視性の制約を理解する。
S13 L7 DNS観測 L7 Whitehat, SRE 2/5 基礎良好 Docker embedded DNSのservice discoveryを観測し、内部偵察と運用依存を理解する。
S14 SREインシデント対応 横断/SRE SRE, Backend, Whitehat 4/5 実務上級 攻撃やDB停止をSLO/incident responseへ接続し、MTTD/MTTRとpostmortemを作る。
S15 ホワイトハット/SRE修了課題 全体 Whitehat, SRE, Backend 4/5 実務上級 S1-S14を組み合わせ、攻撃、検知、修正、運用、報告を一つの成果物にする。
